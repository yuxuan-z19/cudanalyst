# HPC benchmarks

## PolyBench/GPU (poly)

[PolyBench/GPU](https://www.cs.colostate.edu/~pouchet/software/polybench/GPU/index.html) ([sgrauerg/polybenchGpu](https://github.com/sgrauerg/polybenchGpu)) is a collection of [the Polyhedral Benchmark suite](https://www.cs.colostate.edu/~pouchet/software/polybench/) codes (as well as convolution) implemented for processing on the GPU using CUDA, OpenCL, and HMPP (pragma-based compiler).
