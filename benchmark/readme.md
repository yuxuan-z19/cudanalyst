# CUDA Benchmark Suite

A curated collection of CUDA benchmarks for testing, performance evaluation, and analysis. This repository covers classical HPC workloads, DSL/compiler kernels, mini-apps, and emerging LLM-generated CUDA kernels.

```bash
cudanalyst/benchmark
├── app   : CUDA Mini-Apps for small-scale testing
├── cgo   : Benchmarks for DSLs and compiler-generated kernels
├── hpc   : High-Performance Computing (HPC) benchmarks
├── llm   : Benchmarks for LLM-generated CUDA kernels
└── readme.md
```

Each subdirectory contains its own `readme.md` detailing benchmark sources and corresponding BibTeX entries for proper citation.

## Contributing

We welcome contributions, particularly for:

- Adding new CUDA mini-apps or DSL benchmarks.
- Porting HPC workloads.
- Creating additional LLM-driven kernel benchmarks.

Submit a pull request or open an issue to propose new benchmarks.
